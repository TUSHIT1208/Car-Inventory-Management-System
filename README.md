#  Car Inventory Management System (C++)
Car Inventory Management System is a C++ console based application for retailers and suppliers. It supports role based login for Admin, Sales Manager, Sales Person, and Supplier. The system manages products, stock, sales transactions, supplier requests, and reports to ensure secure, accurate, and efficient car inventory and sales operations in C++.

---

## -> System Overview

The system consists of two main sides:

### -> Retailer Side

* **Admin**
* **Staff**

  * Sales Manager
  * Sales Person

### -> Supplier Side

* Supplier

Each role has its own permissions and secure login.

---

## -> User Roles & Features

### -> Admin

* Login / Logout
* Product Management
* Stock Management
* Supplier Management
* Staff Management
* View Reports

### -> Sales Manager

* Login / Logout
* View Products
* Sales Product 
* Send Product Requests to Supplier
* View Sales History
* Respond to Supplier Requests

### -> Sales Person

* Login / Logout
* View Products
* Sell Products
* View Sales History

### -> Supplier

* Login / Logout
* View Product Requests
* Supply Products
* View Supply History
* Product Management

---

## -> System Workflow

1. Admin adds products, staff, and suppliers
2. Sales Manager requests products from supplier
3. Supplier supplies requested products
4. Stock updates automatically
5. Sales Person sells products
6. Admin views reports
